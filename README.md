# Setup Instructions

## 1. Clone Repo
```bash
git clone https://github.com/abhishek-0x49310/learn-ielts-react.git
```
```bash
cd learn-ielts-react
```

## 2. Install Dependencies
```bash
npm install
```

## 3. Run development server
```bash
npm run dev
```

## 4. Run production server
```bash
npm run build
npx serve dist
```
